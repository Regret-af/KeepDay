# 测试用例与验收清单

## 1. 测试策略

优先级：

```text
领域规则单元测试
→ Use Case 测试
→ Repository 测试
→ 页面 Smoke Test
→ 真机通知和文件测试
```

最重要的测试对象：

- 宽限期。
- 暂停期。
- 连续天数。
- JSON 恢复事务。
- 通知规则。

## 2. 习惯管理测试

| 用例 | 预期 |
|---|---|
| 创建合法习惯 | 保存成功，首页展示 |
| 名称为空 | 保存禁用或提示 |
| 名称前后有空格 | 自动 trim |
| 名称超过 30 字 | 保存失败 |
| 名称包含换行 | 保存失败 |
| 创建同名习惯 | 弹出确认 |
| 编辑暂停习惯 | 允许保存 |
| 删除习惯 | 前台不可见，deletedAt 有值 |

## 3. 打卡测试

| 用例 | 预期 |
|---|---|
| 今日未打卡点击打卡 | 新增 today 记录 |
| 今日重复打卡 | 不产生重复记录 |
| 今日取消打卡 | 删除今日记录 |
| 历史补卡 | 新增 makeup 记录 |
| 今天在详情页补卡 | source = today |
| 创建日期前补卡 | 禁止 |
| 未来日期补卡 | 禁止 |
| 暂停期间补卡 | 允许，计入累计 |

## 4. 宽限期测试

| 用例 | 预期 |
|---|---|
| gracePerWeek = 0，漏一天 | missed，连续中断 |
| gracePerWeek = 1，周内第一个漏打 | grace |
| gracePerWeek = 1，周内第二个漏打 | missed |
| gracePerWeek = 2，周内两个漏打 | 均为 grace |
| 创建当周 | 从创建日开始计算 |
| 补卡原 grace 日期 | 状态变 checked，宽限重算 |
| 取消打卡 | 宽限按最新记录重算 |
| 修改宽限期 | 历史状态按新设置重算 |

## 5. 暂停测试

| 用例 | 预期 |
|---|---|
| 暂停 normal habit | status = paused，新增 pause period |
| 恢复 paused habit | status = normal，填充 endDate |
| 暂停习惯不在正常首页列表 | 进入暂停折叠区 |
| 暂停习惯不计入今日进度 | 进度不包含该习惯 |
| 暂停期间不消耗宽限 | 当周 grace 不被暂停日占用 |
| 暂停期间补卡 | checkedDuringPause |
| 暂停期间补卡不计入连续 | best/current 不增加 |
| 全部习惯暂停 | 首页自动展开暂停区 |

## 6. 统计测试

| 用例 | 预期 |
|---|---|
| 创建当天持续天数 | 1 |
| 连续每天打卡 3 天 | currentStreak = 3 |
| 中断且无宽限 | currentStreak = 0 |
| 历史宽限日 | 计入 bestStreak |
| 今天宽限未打卡 | 不计入 currentStreak |
| 暂停 3 天后恢复打卡 | 连续链路从暂停前延续 |
| 取消历史打卡 | best/current 重算 |
| 恢复备份后 | 所有 habit 缓存与记录一致 |

## 7. 首页测试

| 用例 | 预期 |
|---|---|
| 无习惯 | 展示空状态 |
| 有未打卡习惯 | 展示待打卡 |
| 有已打卡习惯 | 展示已完成 |
| 有宽限中习惯 | 展示休息中 |
| 有暂停习惯 | 展示暂停折叠区 |
| 无通知权限 | 展示权限提示 |
| 未打卡、休息中、已完成同时存在 | 排序正确 |

## 8. 通知测试

| 用例 | 预期 |
|---|---|
| 开启提醒且权限允许 | 安排通知 |
| 权限关闭 | 首页提示 |
| 今日已打卡 | 不再提醒 |
| habit 暂停 | 取消后续提醒 |
| habit 恢复 | 重新安排提醒 |
| habit 删除 | 取消提醒 |
| 今日已发送通知后取消打卡 | 不补发已过提醒 |
| 每日重复触发 | 每 habit 每天最多一条 log |

## 9. 备份恢复测试

| 用例 | 预期 |
|---|---|
| 导出数据 | JSON 包含未删除 habit |
| 删除 habit 后导出 | 不包含该 habit 及关联数据 |
| 恢复同版本备份 | 成功覆盖 |
| 恢复高版本备份 | 拒绝恢复 |
| 恢复非法 JSON | 拒绝恢复 |
| 恢复缺必填字段 | 拒绝恢复 |
| 恢复外键缺失 | 拒绝恢复 |
| 恢复中途失败 | 当前数据不变 |
| 恢复成功后 | 重算统计并重建通知 |

## 10. 验收清单

### 10.1 MVP 功能验收

- [ ] 可以创建习惯。
- [ ] 可以编辑习惯。
- [ ] 可以删除习惯。
- [ ] 可以暂停和恢复习惯。
- [ ] 可以今日打卡。
- [ ] 可以取消今日打卡。
- [ ] 可以历史补卡。
- [ ] 可以取消历史打卡。
- [ ] 可以展示月历。
- [ ] 可以展示热力图。
- [ ] 可以设置提醒。
- [ ] 可以导出 JSON。
- [ ] 可以恢复 JSON。

### 10.2 质量验收

- [ ] 关键规则有单元测试。
- [ ] 所有写操作失败时不会产生半成品数据。
- [ ] 恢复失败不会清空当前数据。
- [ ] 删除后数据不再前台展示。
- [ ] Android 真机通知可触达。
- [ ] 文件导入导出在真机可用。

### 10.3 体验验收

- [ ] 首次创建习惯不超过 1 分钟。
- [ ] 首页一键打卡路径清晰。
- [ ] 宽限中和已完成状态不混淆。
- [ ] 暂停习惯容易找回。
- [ ] 危险操作均有二次确认。
