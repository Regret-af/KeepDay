# 页面清单与页面流程

## 1. 页面清单

| 页面 | 路由建议 | MVP | 说明 |
|---|---|---|---|
| 首页 | `/` | 是 | 今日习惯、打卡、进度、暂停区、权限提示 |
| 创建习惯页 | `/habits/new` | 是 | 创建新习惯 |
| 编辑习惯页 | `/habits/:id/edit` | 是 | 编辑已有习惯 |
| 习惯详情页 | `/habits/:id` | 是 | 统计、月历、热力图、补卡、暂停、删除 |
| 设置页 | `/settings` | 是 | 权限、数据导出恢复、关于 |
| 习惯管理页 | `/settings/habits` | 可内嵌 | 设置页中的习惯管理列表 |
| 系统通知权限页 | 系统承载 | 是 | 由系统设置承载 |
| 文件选择器 | 系统承载 | 是 | JSON 恢复 |
| 文件保存/分享面板 | 系统承载 | 是 | JSON 导出 |

## 2. 首页

### 2.1 页面目标

用户打开 App 后立即知道今天有哪些习惯，需要做什么，以及当前完成进度。

### 2.2 页面模块

- 顶部栏：KeepDay、今日日期、创建按钮、设置按钮。
- 通知权限提示。
- 今日完成概览。
- 今日习惯列表。
- 已暂停习惯折叠区。
- 空状态。

### 2.3 今日习惯卡片

卡片内容：

- emoji 或颜色标签。
- 习惯名称。
- 今日状态：未打卡 / 已完成 / 休息中。
- 当前连续天数。
- 累计打卡次数。
- 提醒时间。
- 打卡或已完成按钮。

交互：

- 点击卡片非按钮区域进入详情页。
- 点击打卡按钮立即打卡。
- 点击已完成按钮弹出取消确认。
- 休息中习惯仍允许打卡。

### 2.4 首页排序

```text
未打卡
→ 休息中
→ 已完成
→ 同状态下按 createdAt 升序
```

暂停习惯不进入正常列表，展示在底部折叠区。

## 3. 创建 / 编辑习惯页

### 3.1 页面模块

- 顶部栏：返回、标题、保存。
- 习惯名称输入。
- emoji 输入或快选。
- 颜色色板。
- 宽限期分段选择：0 / 1 / 2 天。
- 提醒开关。
- 提醒时间选择。

### 3.2 表单规则

- 名称自动 trim。
- trim 后不能为空。
- 不允许换行。
- 最多 30 个字符。
- 颜色必选。
- emoji 可选。
- 开启提醒后提醒时间必填。
- 同名习惯允许创建，但必须二次确认。

### 3.3 保存后跳转

- 创建成功：返回首页。
- 编辑成功：返回详情页。

## 4. 习惯详情页

### 4.1 页面模块

- 习惯头部：emoji、颜色、名称、状态、提醒。
- 统计卡片：持续天数、累计打卡、最佳连续、当前连续。
- 月历视图。
- 热力图。
- 日期操作面板。
- 操作区：编辑、暂停/恢复、删除。

### 4.2 日期点击规则

| 日期状态 | 操作 |
|---|---|
| 已打卡 | 可取消打卡，需要确认 |
| 已打卡（暂停期） | 可取消打卡，需要确认 |
| 未打卡 | 可补卡 |
| 宽限日 | 可补卡 |
| 暂停日 | 可补卡 |
| 创建前 | 不可操作 |
| 未来日期 | 不可操作 |

### 4.3 删除流程

```text
点击删除
→ 二次确认
→ 设置 habit.deletedAt
→ 取消通知计划
→ 返回首页
```

## 5. 设置页

### 5.1 页面模块

- 通知权限状态。
- 习惯管理入口。
- 导出 JSON 备份。
- 恢复 JSON 备份。
- 关于 App。
- 本地数据说明。

### 5.2 导出流程

```text
点击导出 JSON 备份
→ 查询未删除数据
→ 生成备份 JSON
→ 调起系统保存/分享面板
→ 成功后提示“备份已导出”
```

### 5.3 恢复流程

```text
点击恢复 JSON 备份
→ 打开文件选择器
→ 读取 JSON
→ 校验格式和版本
→ 展示覆盖恢复确认
→ 事务式恢复
→ 重算统计缓存
→ 重建通知计划
→ 返回首页
```

## 6. 主要页面流程

### 6.1 首次使用

```text
首页空状态
→ 创建习惯
→ 保存
→ 首页展示新习惯
→ 今日打卡
→ 首页进度更新
```

### 6.2 日常打卡

```text
打开首页
→ 查看今日习惯
→ 点击打卡
→ 写入 CheckInRecord
→ 更新统计缓存
→ 更新首页进度
```

### 6.3 补卡

```text
首页进入详情
→ 点击历史未打卡日期
→ 点击补卡
→ 写入 CheckInRecord
→ 重算宽限和统计
→ 更新月历和热力图
```

### 6.4 暂停与恢复

```text
详情页点击暂停
→ 确认暂停
→ habit.status = paused
→ 新增 HabitPausePeriod
→ 取消通知
→ 首页移入暂停区
```

```text
详情页点击恢复
→ habit.status = normal
→ 填充当前 HabitPausePeriod.endDate
→ 重算统计
→ 重建通知
→ 首页进入正常列表
```
