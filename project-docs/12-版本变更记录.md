# 版本变更记录

## 版本规则

KeepDay 使用语义化版本：

```text
主版本.次版本.修订版本
```

示例：

```text
0.1.0
0.2.0
1.0.0
```

## 备份 Schema 版本

MVP 初始备份版本：

```text
1.0.0
```

备份版本与 App 版本不必完全一致，但必须独立维护兼容策略。

规则：

- 同备份版本必须支持恢复。
- 旧备份版本尽量兼容。
- 高备份版本禁止恢复。
